# The Architecture of Inquiry

**问题架构论：AI时代的思维新物种**

> Magic Lamp ThinkTank · Tianhuo Yiwang · 2025 · V1.0

## Abstract

A cognitive operating system for the AI era. Introduces the three-layer penetration model (Phenomenon / Structure / Mental Model), the concept of "definitional entropy reduction," and logical atomization for complex problem-solving. Argues that in an age of answer-overload, the core competency is constructing the right questions.
